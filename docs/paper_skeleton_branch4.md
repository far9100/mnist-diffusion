# 一頁骨架：分支四（負結果短文）

分支四＝CIFAR-100 上 which-FID 不分離 **且** 機制觀察量不複製（三中二不成立）。此時 thesis 於兩個資料集
尺度皆無正面支持，論文為誠實負結果。

## 核心陳述

我們預先登記並執行了「取樣組態應為下游效用而非保真度服務、且由 coverage 機制驅動」之假設檢驗，於 CIFAR-10
與 CIFAR-100 兩尺度。結果：頭條「FID≠效用」在 CIFAR-10 被自家資料反證（which-FID 兩表徵空間皆不分離），
且免訓練選擇器 CaF 敗於更便宜的 FID-min baseline 並具結構性 Pareto 失明；CIFAR-100 上機制觀察量亦未複製。
我們據實報告此負結果。

## 主張順序

1. 動機與假設（H1/H2/H3，`records/2026-07-05-02`）：如實陳述原始主張。
2. 方法：預先登記協定、three-judgment 分立、P0/P1 逐位對帳之嚴謹性（量測可信）。
3. 結果：CIFAR-10 反證（C1、判決三 FID-min 勝 CaF）＋ CIFAR-100 機制不複製。
4. 為何仍有價值：(a) 嚴謹的否證本身（避免文獻中未驗證的順帶宣稱擴散為既定）；(b) 量測方法學貢獻
   （PRDC + FD-DINOv2 + 變異分解 + matched-probe FID-min 對決 + Pareto 失明引理）可重用；(c) 誠實的
   資料集相依性紀錄。
5. 明確不主張：任何普適選擇器、FID≠效用之新發現、CaF 之操作優勢。

## 與 related work 差異化

- 相對隱含宣稱「保真非效用最優」之文獻（DP 擴散、guidance 方法）：本文以預先登記之否證顯示該宣稱在受控
  設定下不穩健，且便宜的 FID-min baseline 常已近最優——為該類宣稱提供一個誠實的反例與量測反駁。
