# A3：block-permutation 敏感度規格（寫碼前凍結）

## 目標

在寫 restricted permutation 程式、跑出第二個 p 之前，凍死塊結構與三態判定規則，杜絕「先跑出 p 再決定方案算不算數」（揭盲後選檢驗）。此為敏感度分析，非替代裁決：-13 標準 permutation 主檢驗（C2a p=0.0188）仍為裁決依據，verdict_pass 不因 block 結果翻案；block 只改 B 判決二為該 pass 掛的 caveat 密度。

## 結果

### 1. 塊結構（凍結）

塊 = gseed 反對角線等價類 {cells : seed+w = const}，30 cells → 14 類：

- **單元素（size 1，不可置換，共 4）**：sum 11=(10,w1)、11.5=(10,w1.5)、14.5=(12,w2.5)、20=(12,w8)。
- **size 2（共 4）**：12={(11,w1),(10,w2)}、12.5={(11,w1.5),(10,w2.5)}、13.5={(12,w1.5),(11,w2.5)}、19={(12,w7),(11,w8)}。
- **size 3（共 6）**：13={(12,w1),(11,w2),(10,w3)}、14={(12,w2),(11,w3),(10,w4)}、15={(12,w3),(11,w4),(10,w5)}、16={(12,w4),(11,w5),(10,w6)}、17={(12,w5),(11,w6),(10,w7)}、18={(12,w6),(11,w7),(10,w8)}。

cell 總數核對：4×1 + 4×2 + 6×3 = 30。

### 2. restricted 置換方案（凍結）

殘差秩只在多元素塊內置換、單元素固定；N=100,000、seed=0（與主檢驗同）。

### 3. 置換空間基數（本輪計算）

restricted 空間 = ∏ (塊大小!) = (1!)⁴·(2!)⁴·(3!)⁶ = 1·16·46656 = **746,496**。
> N=100,000 < 746,496，抽樣不窮舉；空間離散下界 ≈1/746496 ≈1.3e-6，抽樣 p 下界 =1/(N+1) ≈9.9e-6。
> 對 p≈.02–.05 之解析度足夠——**非「幾百種」之失敗情形，基數本身夠大**。

### 4. 三態判定（凍結，對號入座、不臨時解釋）

- 兩者皆 <0.05 = 穩健顯著。
- 主 <0.05 但 block ≥0.05 = 碰撞在撐；B 判決二據實報「顯著依賴共享噪聲、restricted 下不穩」。
- 皆 ≥0.05 = 主檢驗顯著本身是方案 artifact。
- 註：主檢驗已出 p=0.0188，state 3「皆 ≥.05」現不可達；規則仍事前全列，為完整性與制度對稱。

### 5. 約束：敏感度非替代裁決

-13 標準 permutation 主檢驗仍為裁決，verdict_pass 不翻案；block 僅調整 B 判決二之 caveat 密度。協議凍結之裁決不被新寫敏感度翻案，但誠實報告要求把敏感度貼上。

### 6. 結構待決（實作前呈報作者，此為「站不站得住」之真關卡）

基數夠大（§3），但兩個結構問題決定敏感度是否站得住，非基數：

- **測試單位 cell-level(n=30) vs 主檢驗 config-level(n=10)**：塊結構在 cell 層（30 cells）；-13 主檢驗在 per-config 均值（n=10）。block-perm 只能在 cell 層重算 partial ρ，與主檢驗非同 n，§4 三態之「主 vs block」跨 n 比較不對等。建議 apples-to-apples：cell 層同一統計量下跑 free-perm 與 restricted-perm 兩 p 相比（config 層主檢驗 0.0188 獨立保留為裁決）。測試單位待作者定。
- **within-block 不同 w**：同塊 cell 共享初始噪聲但 w 相異、期望值不同；殘差化控 precision＋excess，w 效應經 coverage 流入；within-block 置換打亂不同 w 之 cell，null 定義須確認。

## 後續

作者裁測試單位（cell-level free-vs-restricted，或維持 §4 主-vs-block 框架）後，方實作＋跑；結果對號入座 §4 三態、STOP 呈報。若作者判此敏感度結構上不站得住 → 不硬跑，B 判決二直接寫「碰撞使可交換性受損、restricted permutation 因結構問題無法提供有效敏感度、顯著性穩健性待 CIFAR-100 獨立複製」——誠實承認驗不了，勝於一個粗顆粒假 p。不滾進 A2。
